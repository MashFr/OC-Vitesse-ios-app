- MVVM (Model-View-ViewModel) Architecture

## 📋 Project Info

**Creation Date:** November 5, 2024

**Project Description:**
> This project involves developing a mobile application for the automotive company *Vitesse*, aimed at optimizing candidate management for the Human Resources department. The application enables HR staff to create, edit, and manage a list of candidates, with an option to mark certain candidates as favorites.
